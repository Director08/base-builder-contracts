# Changelog

## v1.0.0 - 2026-03-28

### Added
- BaseToken: ERC-20 token with mint-on-deploy
- BaseNFT: ERC-721 with sequential minting and metadata
- OnchainRegistry: Decentralized key-value store
- PaymentSplitter: Split ETH among configurable payees
- TimeLock: Queue and execute transactions after delay
- MultiSig: Multi-signature wallet with confirmations
- Vault: ETH deposit/withdraw with pause mechanism
- Staking: ERC-20 staking with reward distribution
- SimpleDAO: Onchain proposal and weighted voting
- AirdropDistributor: Batch and claimable token distribution
- CloneFactory: EIP-1167 minimal proxy deployer
- FeeRouter: Protocol fee routing with referrals
- Escrow: P2P escrow with arbiter dispute resolution
- Full test suite for all 13 contracts
- Deployment script for Base mainnet
- Deployed all 13 contracts to Base mainnet
