# Contributing

Contributions are welcome! Here's how to get started:

## Setup
1. Clone the repo
2. Run `npm install`
3. Run `npx hardhat compile`
4. Run `npx hardhat test`

## Guidelines
- Write tests for any new contract
- Follow existing code style
- Use custom errors over require strings where possible
- Keep contracts minimal and self-contained
- Add documentation in the `docs/` folder

## Pull Requests
- One contract per PR
- Include tests
- Update CHANGELOG.md
